# B) PLC 영상 3편 HeyGen 제작 가이드

3편 모두 동일한 아바타·톤·자막 스타일을 유지해야 신뢰가 쌓입니다. 아래 순서(스크립트 → 아바타 → 자막 → 썸네일)를 DAY1/2/3 각각에 반복 적용하세요.

---

## 공통 설정 (3편 동일하게 유지)

**아바타**
- HeyGen Studio Avatar 또는 본인 커스텀 아바타(실제 얼굴 촬영본 학습) 중 선택. 신뢰가 핵심인 퍼널이므로 가능하면 본인 커스텀 아바타 권장.
- 의상: 짙은 네이비 또는 블랙 셔츠/니트 (사이트 골드 컬러(#C9A84C)와 충돌하지 않는 톤)
- 배경: 단색 다크 그레이 또는 블러 처리된 사무실 — 사이트의 블랙 배경과 톤 일치
- 카메라 프레이밍: 가슴 위 클로즈업(bust shot), 정면 고정, 좌우 흔들림 최소
- 제스처 프리셋: "Casual Explaining" 또는 "Confident Presenter" — 손동작 과하지 않게

**음성**
- HeyGen 보이스: 한국어 남성/여성 중 실제 강사 목소리와 톤이 가까운 것 선택
- 속도: 1.0~1.05배 (너무 느리면 이탈률 증가)
- 감정 톤: DAY1은 공감형(느리고 진지), DAY2는 전문가형(또렷하고 빠름), DAY3은 확신형(강하고 힘있게)

---

## DAY1 — "기회" (12분)

**스크립트 타임코드**
| 구간 | 내용 |
|---|---|
| 0:00~1:00 | Hook — "3년 전 한 달 매출 47만 원이었습니다" |
| 1:00~3:00 | 고통 공감 — 좋아요는 있는데 DM이 없는 이유 |
| 3:00~7:00 | 기회 제시 — Jeff Walker PLF 공식 1단계 |
| 7:00~10:00 | 사례 제시 — 2주 만에 첫 판매 ₩1,970,000 |
| 10:00~12:00 | 예고 + CTA — 카카오 채널 추가 안내 |

**HeyGen 제작 순서**
1. 스크립트를 1분 단위로 잘라 HeyGen에 입력 (한 번에 너무 긴 텍스트는 발음 오류 증가)
2. 각 구간 전환마다 아바타 클립을 분리 생성 후 CapCut/Premiere에서 이어붙이기 (지루함 방지)
3. 3:00~7:00 구간(핵심 설명)은 화면에 다이어그램/텍스트 오버레이 추가 — HeyGen "Screen + Avatar" 레이아웃 사용

**자막 (Vrew)**
- 폰트: Pretendard Bold, 크기 42~48px (모바일 세로 기준)
- 색상: 흰색 텍스트 + 검정 outline, 핵심 숫자(₩1,970,000, 2주 등)는 골드(#C9A84C)로 하이라이트
- 위치: 화면 하단 1/3 지점, 2줄 이내로 유지
- 컷당 노출 시간: 최소 1.5초 (Vrew 자동 분할 후 너무 짧은 컷 수동 병합)

**썸네일**
- 구성: 아바타 얼굴(놀란/진지한 표정) + 굵은 숫자 텍스트("월 47만원 → 월 7,800만원")
- 배경: 사이트와 동일한 블랙 + 골드 그라데이션
- 텍스트 최대 2줄, 6~8단어 이내
- DAY 배지 좌상단에 "DAY1" 표시 (연속 시청 유도)

## DAY2 — "전환" (15분)

**스크립트 타임코드**
| 구간 | 내용 |
|---|---|
| 0:00~1:30 | Hook — "오늘은 Claude를 켜고 직접 오퍼를 만들어드립니다" |
| 1:30~10:00 | 핵심 콘텐츠 — Dan Kennedy 오퍼 6요소 실시간 시연 |
| 10:00~13:00 | 성공 사례 — 전환율 4.7%, 업계 평균 3배 |
| 13:00~15:00 | 예고 + CTA — DAY3 자동화 시스템 예고 |

**HeyGen 제작 순서**
1. 1:30~10:00 구간은 실제 Claude 화면 녹화 + 아바타 PIP(작은 화면)로 합성 — HeyGen보다 CapCut의 화면 녹화 오버레이 기능 활용 권장
2. 6요소(핵심가치/희소성/긴박감/보너스/보장/가격)마다 화면에 체크리스트 애니메이션 추가
3. 아바타 목소리와 화면 시연 타이밍을 맞추기 위해 스크립트를 6요소 단위로 나눠 생성

**자막**: DAY1과 동일 스타일. 단, 6요소를 언급할 때마다 해당 단어(핵심가치, 희소성 등)를 골드로 하이라이트.

**썸네일**
- 구성: "Claude 화면 캡처 + 전환율 4.7%" 숫자 강조
- DAY2 배지 + "실시간 시연" 텍스트 배지 추가

## DAY3 — "소유 경험" (18분)

**스크립트 타임코드**
| 구간 | 내용 |
|---|---|
| 0:00~2:00 | Hook — "오늘이 마지막 무료 강의입니다" |
| 2:00~12:00 | 성공 스토리 3편 (박○○/이○○/최○○) |
| 12:00~16:00 | 미래 비전 — "1년 후의 당신을 상상해보세요" |
| 16:00~18:00 | 예고 + CTA — DAY5 특별 발표 예고 |

**HeyGen 제작 순서**
1. 성공 스토리 3편은 각각 별도 클립으로 생성 후, 클립 사이에 실제 결제/후기 스크린샷(모자이크 처리) 삽입
2. 12:00~16:00 미래 비전 구간은 아바타 단독 클로즈업으로 감정선 강조 (편집 컷 최소화)
3. 마지막 CTA 구간은 배경에 카운트다운 텍스트 오버레이 추가 (긴박감 시각화)

**자막**: DAY1/2와 동일. 성공 스토리 구간에서는 이름·직업·결과 숫자를 자막 카드(하단 배너 형태)로 별도 고정 노출.

**썸네일**
- 구성: 3명의 성공 사례를 3분할 콜라주 + "마지막 무료 강의" 배지
- DAY3 배지 + 긴박감 문구("오늘 마감") 추가
